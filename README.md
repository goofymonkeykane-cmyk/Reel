# Reel 🎬

### A free, open source, forever social network for cinema lovers.

---

## What is Reel?

Reel is a cinema-first social platform that combines film tracking, 
taste-based friend matching, direct messaging, watch parties, community 
clubs, and cinema ticket booking — all in one place, completely free, 
with no ads, no paywalls, and no corporate ownership. Ever.

Think of it as what you get if Letterboxd, Moctale, Serializd, 
and a cinema-focused version of Twitter were rebuilt from scratch 
with human connection as the primary feature.

---

## Why does this exist?

Every cinephile uses 3 to 5 different apps and still feels alone.

- Letterboxd has no DMs, no matching, no anime, no watch parties
- Moctale has community but is full of ads and lacks depth
- Serializd tracks series beautifully but has no social layer
- Film Twitter has the conversation but none of the film data

Nobody has built the platform where you find your people through 
the films you love, talk to them, and watch together.

Reel is that platform.

---

## Core Features

- Taste-based cinephile matching (like Tinder but for film taste)
- Full film, series, anime, documentary tracking from every country
- Direct messaging with cinema-native features (share films, watchlists)
- Legal watch parties — everyone logs in with their own OTT account
- Community Clubs (Marvel, Indian Cinema, Anime, World Cinema, K-Drama)
- In-app cinema ticket booking with price comparison across platforms
- Collections and franchise trackers with watch order
- Deep personal statistics — world map, crew breakdown, year in review
- Lock-in on any actor or director — get notified when they announce new work
- Nanogenres — hyper-specific mood tags beyond standard genres
- Release calendar covering every country, language, and platform
- Where to watch — real-time streaming availability by country
- Letterboxd and Serializd import on day one

## Philosophy

Reel is built on a simple belief:

**Cinema belongs to everyone. The platform for cinema lovers should too.**

- Completely free. Forever. For everyone. No feature tiers.
- No advertisements. Ever.
- Open source under AGPL-3.0 license
- Voluntary donations only — like Wikipedia
- Self-hostable — any community can run their own instance
- Transparent public funding page showing every rupee of income and expense
- User data is yours — full export anytime, DMs are encrypted
- If Reel ever shuts down, the code lives on and anyone can continue it

This is not a startup. This is public infrastructure for cinema culture.
Built like VLC. Funded like Wikipedia. Licensed like Linux.

## Technical Direction (for developers)

- Frontend: Next.js 14 + Tailwind CSS + Framer Motion
- Backend: Supabase (PostgreSQL + Realtime + Auth)
- Movie Data: TMDB API (free, 800,000+ titles)
- Media: Cloudinary
- Watch Party Sync: WebSockets (legal — syncs timing only, never content)
- Ticket Booking: BookMyShow API + price comparison layer
- Deployment: Vercel
- License: AGPL-3.0

---

## Monetization — There Is None

Reel has no monetization. It is sustained by:

- Voluntary donations (Wikipedia model)
- Open source contributions (Linux model)
- Infrastructure sponsorships from cultural institutions
- Small affiliate commissions from cinema ticket bookings
  (always shows lowest price first, never influences results)

Every rupee of income is publicly disclosed.
